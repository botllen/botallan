How to install Repository

  1. Go to Kodi Settings → File Manager → Add Source → enter URL https://kodi.botallen.com.
  2. Click “Install From Zip File” and choose the BotAllen source.
  3. Select repository.botallen-X.X.X.zip to install.
  4. Then go to “Install From Repository” → BotAllen Repository → Video Add‑ons.
